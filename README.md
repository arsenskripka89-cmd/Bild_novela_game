# Bild Story Studio

Односторінковий веб-додаток для створення інтерактивних історій у стилі Twine без програмування. Працює повністю у браузері й не потребує білду чи серверної частини.

## Запуск

Відкрийте `index.html` у будь-якому сучасному браузері або запустіть простий статичний сервер (наприклад, `python -m http.server 8000`) у корені репозиторію й перейдіть на `http://localhost:8000`.

## Можливості

- Візуальний редактор сцен із drag-and-drop на полотні та панеллю властивостей.
- Створення виборів із умовами, зміною змінних і переходами між сценами.
- Робота з мультимедіа: фон, зображення, відео, аудіо в межах кожної сцени.
- Попередній перегляд із живими змінними й проходженням різних гілок.
- Експорт у HTML (самодостатній переглядач) та JSON, імпорт проєктів і шаринг через посилання (дані кодуються в URL).
- Автозбереження в `localStorage` та можливість редагування стартової сцени.

## Експорт та спільний доступ

- **Експорт HTML:** генерує автономний переглядач історії з усіма сценами, виборами та змінними.
- **Експорт/імпорт JSON:** зручно для резервних копій чи подальшого використання у сторонніх інструментах.
- **Посилання на проєкт:** кнопка «Скопіювати посилання на проєкт» додає дані історії в URL, що дозволяє швидко поділитися чернеткою.
